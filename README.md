## Pulse research review

The public-safe review package for the Pulse crypto-trading project is in
[`pulse_research/`](pulse_research/README.md). It contains the hypothesis
register, failed-test results, preregistration/audit snapshots, and the
reproducible research/dashboard code snapshots.

Raw market dumps, runtime logs, databases, credentials, server access details,
and paid-data samples are intentionally excluded.
